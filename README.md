# WordBucket

- 英単語学習アプリです
- [Demo](https://word-bucket.narulab.xyz/)あります
- バックエンドのリポジトリは[こちら](https://github.com/na-ah/WordBucketAPI)です

## 特徴

- 単語ごとに学習回数、正答率、所要時間を記録
- フラッシュカードで学習可能
- 英文読解モード搭載

  - 登録済の単語はステータスに応じて色分け表示可能
  - Google 検索や DeepL 翻訳が可能
